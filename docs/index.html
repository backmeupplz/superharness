How it works

Three steps. No babysitting required.

1

Plan

Tell SuperHarness what you want to build in one sentence. It creates a staged roadmap automatically and assigns tasks to parallel agents — no manual decomposition needed.

2

Away

Step away or go to sleep. A team of AI agents works in isolated git worktrees simultaneously. Smart supervision handles stalls, loops, and safe permission prompts without waking you.

3

Present

Come back to a full debrief of what was built, what's in progress, and any decisions waiting for you. Review, decide, and optionally kick off the next stage.

What you get

Four things that matter.

🗺️

Autonomous Planning

Describe a goal. SuperHarness breaks it into a DAG of tasks, assigns the right model to each, and kicks them all off in parallel — no planning required from you.

⚡

Parallel Execution

Every agent works in its own isolated git worktree. No conflicts, no waiting. A feature that takes one agent 8 hours takes a team of 4 about 2.

👁️

Smart Supervision

Background monitor detects stalled agents and recovers automatically. Loop detection catches agents spinning in circles. Only decisions that actually need you are queued for your return.

📋

Full Debrief

When you run present, you see a timestamped event log, task completion status, and a list of any decisions the team couldn't make without you.

opencode alone vs. superharness

SuperHarness replaces opencode. You never need to run opencode directly again.

opencode alone
  • One agent, one task at a time
  • Sequential execution — you wait
  • You manage the terminal
  • You answer every prompt
  • No memory of what happened
superharness
  • Team of agents, parallel stages
  • Autonomous — works while you sleep
  • No terminal management needed
  • Safe prompts auto-approved; only real decisions queued
  • Full debrief when you return

Quick start

Four commands. One night. Shipped feature.

terminal
$cargo install superharness
$cd your-project
$superharness plan "build a REST API with JWT auth"
$superharness away
# go to sleep
$superharness present
